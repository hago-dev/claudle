Claudle 🐩 — AI 사용량 메뉴바 앱 (macOS 전용)
================================================

■ 설치 (터미널 1줄 필요)

1) Claudle.app 을 "응용 프로그램(/Applications)" 폴더로 드래그.
2) "터미널" 앱을 열고 아래 한 줄을 붙여넣어 실행:

     xattr -dr com.apple.quarantine /Applications/Claudle.app

   (슬랙/메일/드라이브로 받은 앱은 macOS 가 격리 표시를 붙여 막습니다.
    이 한 줄이 그 표시를 지웁니다. AirDrop/USB 로 받았다면 생략 가능.)
3) Claudle.app 더블클릭 → 실행됩니다.
4) 화면 상단 메뉴바에 🐩 아이콘이 나타납니다. (Dock 아이콘 없음)
   - 첫 실행 시 "키체인 접근 허용" 창이 뜨면 "항상 허용" 클릭.


■ 전제조건

- 이 맥에 Claude Code 가 설치·로그인되어 있어야
  구독 한도 패널이 표시됩니다.
- 사용량 데이터는 각자 맥의 ~/.claude 로그를 기준으로 집계됩니다.


■ 사용법

- 메뉴바 아이콘 텍스트 = 현재 세션 사용률 + 리셋까지 남은 시간.
  강아지는 Claude 를 실제 사용 중일 때 달립니다(유휴 시 정지).
- 아이콘에 마우스를 올리면 세션 한도가 언제 초기화되는지
  한국 시간(KST)으로 보여줍니다.
- 아이콘 우클릭 → "대시보드 열기": 일별/모델별/프로젝트별 상세 + 기간별 조회.
- 종료: 아이콘 우클릭 → "종료".


■ 참고

- 사내 배포용 ad-hoc 빌드 — Apple 공증 전이라 위 xattr 한 줄이 필요합니다.
- 유니버설 빌드 — 인텔 맥 · 애플실리콘 맥 모두 지원. macOS 11.5 이상.
